Web Accessibility Checklist
User Characteristic Accessible design tip
Unable to see Code all images with ALT text
Unable to preceive colors Avoid using color alone to convey information
Unable to use mouse Be sure all website features can be accessed using keyboard
Unable to hear Add capions to multimedia
Prone to having seizures Avoid content that flashes
Easily distractible Keep the design simple

Accessibility Review of the Southwest Career and Technical Academy Home Page

Errors:

  • The banner role that is used is unnecessary
  • Multiple sections lack headings
  • Multiple tags used are obsolete
  • Bad value for true
  • Multiple attributes are not allowed at the point they are placed in

Ideas for solution:

  • Go through and remove obsolete tags
  • Add identifying headings to sections that are missing it
  • Do an overall update to the tags that are being used
